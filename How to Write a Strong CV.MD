# How to Write a Strong CV (Summary, Keywords, and Tips)

## 1. How to Write a Professional Summary

Your CV summary is a short paragraph (3–5 lines) at the top of your CV that highlights your experience, skills, and career goals. [uk.indeed](https://uk.indeed.com/career-advice/cvs-cover-letters/cv-summary-examples)

### Structure:
- **Who you are** (job title or field)
- **Years of experience**
- **Key skills or achievements**
- **Career goal or value you bring**

### Example (Data Analyst Focus):
> Entry-level Data Analyst with ITI Power BI track and SQL expertise from freelance projects. Developed ETL dashboards for sports analytics, boosting efficiency 30%. Passionate about data warehousing and Python for scalable insights in dynamic teams.

### Tips:
- Keep it **concise and clear** (under 100 words)
- Tailor it for **each job** using role-specific keywords
- Use **strong action words** like optimized, analyzed, engineered
- Avoid personal pronouns (I, me, my)

## 2. How to Search for Keywords

Keywords are important because many companies use **ATS (Applicant Tracking Systems)** to scan CVs—90% of large firms rely on them. [atscvchecker](https://www.atscvchecker.pro/blog/best-ats-resume-checker-2026/)

### Where to Find Keywords:
- Job descriptions on LinkedIn
- Company career pages
- Industry postings (e.g., data analyst roles)

### Steps:
1. Read the job description carefully
2. Highlight repeated skills and requirements (SQL, Power BI, ETL)
3. Note important tools, technologies, and qualifications
4. Add these keywords naturally into your CV

### Example:
If the job mentions:
- "Advanced SQL queries"
- "Power BI dashboards"
- "Data engineering"

#### Include them like:
- Engineered advanced SQL queries (CTEs/views) for Power BI dashboards in data engineering projects

### Tips:
- Use **exact phrases** from the job post for ATS match
- Don’t overstuff keywords (keep it natural, 70-80% coverage)
- Include keywords in:
  - Summary
  - Skills section
  - Work experience
- **Free Keyword Tool**: Paste job description into [Himalayas Keyword Finder](https://himalayas.app/job-description-keyword-finder) or [ResumeUp Keyword Finder](https://resumeup.ai/job-description-keyword-finder) for instant extraction. ## 3. Advice for Writing a Strong CV

### General Tips:
- Keep it **1 page** for entry-level (2 max for seniors)
- Use a **clean layout** (reverse chronological)
- Choose **professional fonts** (Arial/Calibri 10-12pt)

### Content Tips:
- Focus on **achievements, not duties**
- Use **numbers and results**:
  - ❌ Handled data analysis
  - ✅ Analyzed datasets in Excel/Power BI, cutting report time 40%
- Start bullets with **action verbs**: Analyzed, Built, Optimized, Visualized

### Formatting Tips:
- Bullet points over paragraphs
- Consistent spacing/alignment
- Save as PDF (ATS-friendly)

### Common Mistakes to Avoid:
- Typos/grammar errors
- Generic CVs for all jobs
- Irrelevant info (hobbies unless relevant)
- Unprofessional email


## Final Tip

Always customize your CV for each job, run it through an ATS checker, and align with LinkedIn for data roles in Egypt.
